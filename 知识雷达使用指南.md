# 知识雷达功能 - 使用指南

## 已实现的核心功能

### 1. 试卷扫描与错题识别
**服务类**: `ExamPaperAnalysisService.kt`
```kotlin
// 使用方法
val service = ExamPaperAnalysisService(ocrService, qwenService)
val result = service.analyzeExamPaper(imagePath)

if (result.success) {
    // result.wrongQuestions 包含所有识别的错题
    result.wrongQuestions.forEach { question ->
        println("题目: ${question.questionText}")
        println("答案: ${question.correctAnswer}")
        println("知识点: ${question.knowledgePoints}")
    }
}
```

### 2. 批量添加错题到错题本
**仓库类**: `RadarRepository.kt`
```kotlin
// 使用方法
repository.batchAddWrongQuestions(
    wrongQuestions = result.wrongQuestions,
    imagePath = imagePath
)
```

### 3. AI举一反三生成测试题
```kotlin
// 为某个错题生成3道类似题目
val similarQuestions = service.generateSimilarQuestions(
    wrongQuestion = wrongQuestion,
    count = 3
)
```

### 4. 知识点掌握度分析
**分析器**: `KnowledgeMasteryAnalyzer.kt`
```kotlin
val analyzer = KnowledgeMasteryAnalyzer()
val wrongQuestions = repository.getAllWrongQuestions()
val subjectMastery = analyzer.analyzeSubjectMastery(wrongQuestions)

// 查看各学科掌握度
subjectMastery.forEach { (subject, mastery) ->
    println("$subject: ${mastery.masteryScore}分")
    println("错题数: ${mastery.wrongCount}")
    println("薄弱知识点: ${mastery.weakKnowledgePoints}")
}
```

### 5. 雷达图可视化
**组件**: `RadarChart.kt`
```kotlin
@Composable
fun MyRadarScreen() {
    val data = mapOf(
        "数学" to 85f,
        "物理" to 70f,
        "化学" to 90f,
        "英语" to 75f
    )

    RadarChart(
        data = data,
        maxValue = 100f
    )
}
```

## 完整使用流程示例

```kotlin
@Composable
fun RadarFeatureDemo() {
    val scope = rememberCoroutineScope()
    val context = LocalContext.current

    // 1. 初始化服务
    val ocrService = OCRService()
    val qwenService = QwenAIService()
    val examService = ExamPaperAnalysisService(ocrService, qwenService)
    val repository = RadarRepository(...)

    // 2. 扫描试卷
    Button(onClick = {
        scope.launch {
            // 分析试卷
            val result = examService.analyzeExamPaper(imagePath)

            if (result.success) {
                // 批量添加错题
                repository.batchAddWrongQuestions(
                    result.wrongQuestions,
                    imagePath
                )

                // 为第一道错题生成举一反三题目
                val similar = examService.generateSimilarQuestions(
                    result.wrongQuestions.first()
                )
            }
        }
    }) {
        Text("扫描试卷")
    }

    // 3. 显示掌握度雷达图
    val wrongQuestions by repository.getAllWrongQuestions()
        .collectAsState(initial = emptyList())

    val analyzer = KnowledgeMasteryAnalyzer()
    val mastery = analyzer.analyzeSubjectMastery(wrongQuestions)

    val radarData = mastery.mapValues { it.value.masteryScore }
    RadarChart(data = radarData)
}
```
