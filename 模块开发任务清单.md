## 🔄 每日工作流程

### 早上（开始工作）

1. **同步代码**
   ```bash
   git checkout develop
   git pull origin develop
   git checkout feature/你的分支
   git merge develop
   ```

2. **查看任务**
   - 检查任务清单
   - 明确今日目标

3. **开始编码**

### 下午（提交代码）

1. **测试功能**
   - 运行应用
   - 测试新功能
   - 修复Bug

2. **提交代码**
   ```bash
   git add .
   git commit -m "feat: 实现XX功能"
   git push origin feature/你的分支
   ```

3. **更新进度**
   - 更新任务清单
   - 同步团队进度
