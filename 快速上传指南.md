## 方法二：手动执行（如果脚本失败）

### 步骤 1: 在 GitHub 上创建仓库

1. 访问 https://github.com/huohua51
2. 点击右上角 "+" → "New repository"
3. 仓库名：`EduSmart`（或你喜欢的名字）
4. 选择 Public 或 Private
5. **不要**勾选 "Initialize with README"
6. 点击 "Create repository"

### 步骤 2: 在项目目录初始化 Git（如果还没有）

```powershell
# 确保在项目目录中
cd "C:\Users\18335\Desktop\软件创新大赛"

# 初始化 Git（如果还没有）
git init

# 添加所有文件
git add .

# 创建初始提交
git commit -m "feat: 初始提交 - EduSmart 智能教育助手项目"
```

### 步骤 3: 连接到 GitHub

```powershell
# 添加远程仓库（替换为你的仓库名）
git remote add origin https://github.com/huohua51/EduSmart.git

# 验证
git remote -v
```

### 步骤 4: 推送代码

```powershell
# 推送主分支
git push -u origin master
```

**如果遇到认证问题：**
- 使用 Personal Access Token 替代密码
- 或者配置 SSH 密钥

### 步骤 5: 创建 develop 分支

```powershell
# 创建并切换到 develop
git checkout -b develop

# 推送 develop
git push -u origin develop
```

### 步骤 6: 创建模块分支


## 常见问题

### Q: "fatal: remote origin already exists"
A: 删除后重新添加：
```powershell
git remote remove origin
git remote add origin https://github.com/huohua51/EduSmart.git
```

### Q: "Authentication failed"
A: 
1. 使用 Personal Access Token：
   - GitHub → Settings → Developer settings → Personal access tokens
   - 生成新 token，勾选 `repo` 权限
   - 使用 token 作为密码

2. 或配置 SSH：
   ```powershell
   # 生成 SSH 密钥
   ssh-keygen -t ed25519 -C "your_email@example.com"
   
   # 添加 SSH 密钥到 GitHub
   # 复制 ~/.ssh/id_ed25519.pub 内容到 GitHub → Settings → SSH keys
   ```

### Q: "fatal: pathspec did not match any files"
A: 确保在正确的项目目录中，并且文件确实存在。
