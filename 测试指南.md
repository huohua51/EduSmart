# 腾讯云开发注册登录测试指南

## 📱 测试步骤

### 1. 运行应用

#### 方法一：Android Studio
1. 打开 Android Studio
2. 连接 Android 设备或启动模拟器
3. 点击运行按钮（绿色播放图标）或按 `Shift + F10`

#### 方法二：命令行
```bash
# 编译并安装到设备
./gradlew installDebug

# 或者直接运行
./gradlew installDebug && adb shell am start -n com.edusmart.app/.MainActivity
```

### 2. 查看日志（重要！）

#### 在 Android Studio 中查看 Logcat
1. 打开底部 **Logcat** 标签
2. 过滤标签：输入 `CloudBaseService` 或 `CloudBaseUserRepository`
3. 或者查看所有日志：选择 `No Filters`

#### 使用命令行查看日志
```bash
# 查看所有日志
adb logcat

# 只查看应用相关日志
adb logcat | grep -E "CloudBaseService|CloudBaseUserRepository|EduSmart"

# 实时查看并过滤
adb logcat -s CloudBaseService:D CloudBaseUserRepository:D
```

### 3. 测试注册功能

#### 步骤：
1. **打开应用** - 会自动显示登录界面
2. **切换到注册模式** - 点击"还没有账号？立即注册"
3. **填写信息**：
   - **用户名**：例如 `测试用户` 或 `testuser`
   - **邮箱**：例如 `test@example.com` 或 `your-email@163.com`
   - **密码**：例如 `123456` 或 `password123`
4. **点击"注册"按钮**
5. **观察日志输出**，应该看到：
   ```
   CloudBaseService: ========== 开始注册 ==========
   CloudBaseService: 环境ID: edusmart-dev-3gqo04ike66344ea-1327750873
   CloudBaseService: 邮箱: test@example.com
   CloudBaseService: 用户名: 测试用户
   CloudBaseService: API URL: https://edusmart-dev-3gqo04ike66344ea-1327750873.ap-shanghai.app.tcloudbase.com/api/cloudbase-auth
   CloudBaseService: 发送注册请求...
   CloudBaseService: 收到响应，状态码: 200
   CloudBaseService: ✅ 注册成功
   ```

#### 成功标志：
- ✅ 界面显示"注册成功"提示
- ✅ 自动跳转到主界面
- ✅ 日志显示 `✅ 注册成功`

#### 失败排查：
- ❌ **状态码 404**：检查云函数是否部署，HTTP 触发器路径是否为 `/cloudbase-auth`
- ❌ **状态码 400**：检查请求参数格式，查看日志中的响应内容
- ❌ **网络超时**：检查网络连接，确认环境ID和区域配置正确
- ❌ **INVALID_ENV**：检查环境ID是否正确，是否启用了 HTTP 访问服务

### 4. 测试登录功能

#### 步骤：
1. **如果已注册**，直接使用注册时的邮箱和密码登录
2. **如果未注册**，先完成注册步骤
3. **填写登录信息**：
   - **邮箱**：使用注册时的邮箱
   - **密码**：使用注册时的密码
4. **点击"登录"按钮**
5. **观察日志输出**，应该看到：
   ```
   CloudBaseService: ========== 开始登录 ==========
   CloudBaseService: 邮箱: test@example.com
   CloudBaseService: API URL: https://edusmart-dev-3gqo04ike66344ea-1327750873.ap-shanghai.app.tcloudbase.com/api/cloudbase-auth
   CloudBaseService: 发送登录请求...
   CloudBaseService: 收到响应，状态码: 200
   CloudBaseService: ✅ 登录成功
   ```

#### 成功标志：
- ✅ 界面显示"登录成功"提示
- ✅ 自动跳转到主界面
- ✅ 底部导航栏显示（首页、拍照识题等）

### 5. 测试错误情况

#### 测试邮箱已存在：
- 使用已注册的邮箱再次注册
- 应该显示"邮箱已被注册"错误

#### 测试密码错误：
- 使用正确的邮箱，但输入错误的密码
- 应该显示"邮箱或密码错误"

#### 测试空字段：
- 不填写任何信息直接点击按钮
- 按钮应该被禁用（灰色）

## 🔍 常见问题排查

### 问题 1：应用闪退
**排查步骤：**
1. 查看 Logcat 中的红色错误日志
2. 检查是否有 `NullPointerException` 或 `ClassNotFoundException`
3. 确认 `SDKConfig.kt` 中的环境ID配置正确

### 问题 2：网络请求失败
**排查步骤：**
1. 检查设备网络连接
2. 确认环境ID格式：`edusmart-dev-3gqo04ike66344ea-1327750873`
3. 确认区域配置：`ap-shanghai`
4. 在浏览器中访问：`https://edusmart-dev-3gqo04ike66344ea-1327750873.ap-shanghai.app.tcloudbase.com/api/cloudbase-auth`
   - 应该返回 JSON 错误（这是正常的，因为需要 POST 请求）

### 问题 3：云函数执行失败
**排查步骤：**
1. 登录 [腾讯云开发控制台](https://console.cloud.tencent.com/tcb)
2. 进入"云函数" → "cloudbase-auth"
3. 查看"日志查询"，检查是否有错误
4. 确认数据库集合已创建：
   - `users` 集合
   - `tokens` 集合

### 问题 4：密码加密错误
**排查步骤：**
1. 确认云函数已安装 `bcryptjs`：
   ```bash
   cd cloudbase-auth
   npm install
   ```
2. 重新部署云函数
3. 查看云函数日志，确认没有 `bcryptjs` 相关的错误
